# 📱 TransGate - Android Management App

TransGate es la solución móvil del ecosistema de gestión Transdoor. Esta aplicación permite a los usuarios gestionar registros en tiempo real directamente desde sus dispositivos Android, garantizando movilidad y acceso inmediato a la información del servidor.

## 🚀 Funcionalidades Principales
* **Sincronización en Tiempo Real:** Conexión directa con el backend mediante API REST.
* **Interfaz de Usuario (UI):** Diseño intuitivo desarrollado con Android SDK para una gestión eficiente.
* **Consumo de Datos JSON:** Procesamiento de datos dinámicos obtenidos del servidor SQL.
* **Gestión de Sesiones:** Sistema de login seguro para administradores.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Java (Android SDK).
* **Comunicación:** Integración con API REST (Transdoor Backend).
* **Gestión de Dependencias:** Gradle.
* **Formato de datos:** JSON.

## 📁 Estructura del Repositorio
* `/app/src/main`: Contiene el código fuente Java y los layouts XML.
* `gradle/`: Archivos de configuración del entorno de compilación.
